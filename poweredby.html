This site is powered by the following:

  • Nginx, as webserver and reverse proxy

  • nginx-rtmp-module, as rtmp stream ingest

  • ngx-fancyindex, as directory index

  • Gitea, as git server

  • Photofloat, as image gallery

  • hls.js, as video web player

  • Syncthing, as file synchronisation tool

  • Wireguard, as encrypted endpoint transport

  • BIRD, as internal routing daemon

  • GoTTY, as terminal broadcast tool

  • Mailcow, as mailserver

  • Hugo, as blog static site generator

  • Fiche, as command-line pastebin

  • LetsEncrypt, as certificate authority

  • Bitwarden, as password manager

  • HTML5UP, as website designer

  • Debian, as host operating system
